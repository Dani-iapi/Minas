Analise

O carro:

1- o carro deve ser movido a energia solar.
2- o carro deve ter um motor hipersônico para todas as viagens serem rapidas.
3- todos os carro devem ser confortavel para os clientes.

O sistema:
1- o sistema deve ter IA, para controlar o veiculo.
2- o sistema deve gerar um relatorio da corrida para o passageiro.
3- o sistema deve identificar o carro mais proximo ao passageiro
para efetuar a viagens.
4- o sitema deve mostrar o trajeto e o tempo de chegada ao destino.
5- o Sistema so podera aceitar pagamento via bitcoins ou debito automatico
antes de começar a corrida.
6- o sistema deve ser seguro para evitar acidentes enquanto estiver voando.
7- o sistema precisa ser o mais facil possivel para o usario solicitar o carro.
8- o sistema deve calcular o preço para o destino.
9- o sistema deve avaliar o melhor lugar possivel para efetuar
   o pouso para evitar acidentes e nao causar transito.
10-o sistema deve ter identificação do passageiro e do carro.
